# Функції додатку "Генератор паролів з додатковими налаштуваннями"

1. **Генерація паролів**  
   Користувач може створювати випадкові паролі, задаючи довжину та типи символів (великі літери, малі літери, цифри, спеціальні символи).

2. **Налаштування параметрів**  
   Можна включати/виключати певні символи, а також генерувати кілька паролів одночасно.

3. **Оцінка складності**  
   Система визначає надійність пароля (слабкий, середній, сильний, дуже сильний) на основі довжини та різноманітності символів.

4. **Збереження паролів**  
   Згенеровані паролі можна зберігати у базі даних разом із параметрами генерації та датою створення.

5. **Перегляд історії**  
   Користувач може переглядати раніше згенеровані паролі, копіювати їх або видаляти.
